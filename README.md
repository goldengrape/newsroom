# Newsroom

[中文说明](./README.zh-CN.md)

Newsroom is a Gemini-powered daily tech intelligence pipeline. It fetches RSS feeds, filters and translates selected items, publishes a static GitHub Pages briefing, and supports iterative refinement of the plain-text filter profile from local feedback exports.

## Project Layout

```text
src/newsroom/        Python source package
tests/               Pytest suite
data/                Filter profile, raw RSS payloads, feedback exports, learned outputs
docs/                Static site for GitHub Pages
program_docs/        Internal engineering documentation
.github/workflows/   Automation for the daily pipeline and Pages deployment
```

## Quick Start

1. Install dependencies with uv:
   ```bash
   uv sync --dev
   ```
2. Create a local `.env` file:
   ```dotenv
   GEMINI_API_KEY=your_api_key
   ```
3. Fetch RSS news:
   ```bash
   uv run newsroom-fetch
   ```
4. Filter and publish the briefing:
   ```bash
   uv run newsroom-filter
   ```
5. Run the test suite:
   ```bash
   uv run pytest
   ```

## CLI Commands

- `uv run newsroom-fetch`
  Fetch RSS feeds into `data/raw_news.json`.
- `uv run newsroom-filter`
  Screen, rank, translate, and write the static-site payload to `docs/data/news.json`.
- `uv run newsroom-learn-filter --feedback data/feedback/feedback-YYYY-MM-DD.json`
  Learn from exported feedback JSON and generate a candidate updated profile in `data/`.
- `uv run newsroom-gemini-smoke`
  Smoke test the configured Gemini API key.

## Data Files

- `data/FILTER_PROFILE.md`
  The current plain-text filtering protocol.
- `data/raw_news.json`
  The latest fetched RSS payload.
- `data/feedback/*.json`
  Local feedback exports from the static site. These are ignored by Git by default.
- `data/FILTER_PROFILE.updated.md`
  Candidate profile generated by the learning script.
- `data/FILTER_PROFILE.learning_report.md`
  Diagnostic output and raw model response from the learning step.

## GitHub Automation

- `.github/workflows/daily-news.yml`
  Runs the daily RSS fetch, filtering pipeline, and commits refreshed static data.
- `.github/workflows/deploy-pages.yml`
  Deploys `docs/` to GitHub Pages on push to `main`.
